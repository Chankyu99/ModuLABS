# 3. 다양한 데이터 활용법

## 학습 내용

- 특정 데이터 조건과 별칭 
    - 특정 데이터 조건주기 (LIKE, Wildcard)
    - 별칭 사용하기 (Alias)

- 데이터 정렬과 중복 제거
    - 데이터 정렬하기 (ORDER BY)
    - 중복 제거하기 (DISTINCT)

- 조건문 활용하기 
    - 단일 조건문 (IF)
    - 다중 조건문 (CASE WHEN ... THEN)

- 데이터 타입 변환하기 (CAST)

## 3-1. 특정 데이터 조건과 별칭

### LIKE

- 조회 조건 값이 명확하지 않을 때 특정 내용을 찾기 위해 사용
- 조건에 문자나 숫자 포함가능
- 주로 Wildcard와 함께 사용

### Wildcard

- % : 조건을 포함하는 모든 문자를 의미
- _ : 한 글자를 의미

> _ 의 개수에 따라 뒤에 오는 문자열의 개수를 정할 수 있다

### Alias

임시로 컬럼명을 변경할 때 사용함 (기존 컬럼명은 변경되지 않음)

- 컬럼명 뒤에 AS 키워드를 사용하고 원하는 컬럼명을 입력
- AS 키워드는 생략 가능
- 특수문자, 대소문자, 공백 -> " " 사용
- SELECT, FROM 절에서만 사용 가능

## 3-2. 데이터 정렬과 중복 제거

### 데이터 정렬하기 (ORDER BY)

- WHERE 절 다음에 위치한다
- 오름차순 : 컬럼명 뒤에 ASC (기본값)
- 내림차순 : 컬럼명 뒤에 DESC
- ,로 여러 컬럼을 선택해 첫번째 기준에서 같은 값이 있다면 두번째 기준을 적용한다

### 중복 제거하기 (DISTINCT)

- 출력에서 고유한 값을 반환
- SELECT 절 내 모든 컬럼을 고려하여 중복 제거
- 이후에 배울 집계함수와 같이 사용 가능

## 조건문 활용하기

### 단일 조건문 (IF)

- IF(조건, 참일 때 값, 거짓일 때 값)
- SELECT, WHERE 절에서 사용 가능
- 조건이 참이면 첫번째 값을 반환하고, 거짓이면 두번째 값을 반환
- 조건이 참이면 첫번째 값을 반환하고, 거짓이면 두번째 값을 반환

### 다중 조건문 (CASE WHEN ~ THEN)

- CASE WHEN 조건1 THEN 값1
       WHEN 조건2 THEN 값2
       ELSE 만족하는 조건이 없을때 값
  END

- WHEN의 조건을 순차적으로 체크하고 먼저 만족하는 조건의 값을 반환한다
    - WHEN과THEN은 한 쌍으로 존재
    - WHEN과THEN은 여러 개가 존재할 수 있다
    - ELSE는 생략 가능
    - ELSE가 생략되어 있고 만족하는 조건이 없을때 NULL을 반환한다
    - CASE는 END로 마무리한다

### CAST 

- CAST(값 AS 데이터타입)
- 데이터의 타입을 변환하는 함수

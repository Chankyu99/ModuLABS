# 2. 데이터 다루기

## 학습 내용

- 쿼리문 작성과 데이터 필터링
    - 쿼리문 작성(SELECT, FROM)
    - 전체 데이터 필터링(WHERE)

- 비교 연산자와 논리 연산자
    - 비교 연산자 : = , != , < , > , <= , >= , BETWEEN , IN , LIKE , IS NULL , IS NOT NULL
    - 논리 연산자 : AND , OR , NOT
    - 산술 연산자 : + , - , * , / , %

## 2-1. 쿼리문 작성과 데이터 필터링

### 쿼리문 작성하기 (SELECT, FROM)

- 전체 조회는 *로 표시한다

```sql
SELECT 컬럼 FROM 테이블명;
```

### 전체 데이터 필터링하기 (WHERE)

- WHERE은 FROM 절 다음에 위치한다
- 작동 순서는 FROM -> WHERE -> SELECT 순이다
- 조건에 연산자를 같이 사용해야 한다

```sql
SELECT 컬럼 FROM 테이블명 
WHERE 조건;
```

## 2-3. 비교 연산자

- = : 같다
- !=(<>) : 같지 않다
- > : 크다
- < : 작다
- >= : 크거나 같다
- <= : 작거나 같다

## 2-4. 논리 연산자

- AND : 두 조건이 모두 참일 때
- OR : 두 조건 중 하나라도 참일 때
- NOT : 조건을 부정할 때
- IN : 여러개 값 중 하나일 때
- NULL : 값이 없을 때
- BETWEEN : ~사이

## 2-5. 산술 연산자

- () : 괄호 안의 연산을 먼저 수행
- *,/,% : 곱하기, 나누기, 나머지
- +,- : 더하기, 빼기

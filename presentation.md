# 🚀 4. 핵심 기술 1: 데이터 증강(Augmentation) 파이프라인

단순한 RAG를 넘어, **'검색 미적중(Miss)'** 한계를 돌파 구축 모델.

- **원인**: 사용자는 "보배(보조배터리)", "미숫가루" 등 비정형 일상 단어를 쓰지만, 실제 관세 규정 문서는 "리튬이온", "농산물/가공식품" 등 딱딱한 어휘로 작성됨.
- **해결**: `GPT-4o-mini`를 활용해 파이프라인 전처리 과정에서 84개의 원본 문서마다 **연관 동의어와 예상 키워드 15개 이상을 오프라인에서 무수히 증강**하여 강제로 병합.
- **효과**: 문맥 이질감을 제거하고 Vector DB의 검색(Retrieval) 적중률을 한계점까지 끌어올림.

<!--
[대본]
첫 번째 차별점은 검색 적중률을 극적으로 높인 '데이터 증강'입니다. 유저는 "보배"나 "고데기" 같이 일상어를 쓰지만 규정집에는 "리튬이온", "가열식 개인용품" 과 같이 적혀 있어 기존 RAG 방식으로는 매핑이 엇나가는 현상이 있었습니다. 저희는 이를 해결하고자 LLM으로 각 문서에 수십 개의 일상 검색어 키워드를 역으로 증강시켜 DB의 포용력을 대폭 확장했습니다.

# ⚡ 5. 핵심 기술 2: 속도 한계 돌파 (LLM 내부 해킹)

LLM의 무거운 추론 과정을 걷어내어 **답변 대기 시간(TTFT) 60% 단축** 성공.

- **기존 문제**: 한 번의 대답을 얻기까지 평균 약 40초의 극심한 대기 시간 발생(병목).
- **최적화 1 (One-shot Prompting)**: 슬롯 추출 API와 DB 매핑 API를 분리했던 로직을 하나로 합쳐 네트워크 왕복(Round-Trip) 횟수 1단축.
- **최적화 2 (`reasoning_effort="low"`)**: 가장 최신 모델인 GPT-5의 히든 추론 비용 파라미터를 강제 통제. 불필요한 수학적/논리적 망상을 잘라냄.
- **압도적 결과**: 파이프라인 종합 처리 시간 평균 **40초 ➡️ 15초 내외 (약 2.5배 쾌속화)**.

<!--
[대본]
두 번째 핵심 역량은 철저한 '레이턴시(속도) 단축'입니다. 챗봇의 기본 생명은 속도입니다. 저희는 전체 파이프라인의 API 왕복 로직을 싱글 프롬프팅으로 압축하고, 추가로 GPT-5 계열 모델의 내부 추론 연산 스위치를 강제로 낮추었습니다(reasoning effort=low). 문서 기반 RAG는 과도한 추론이 필요 없음을 증명해낸 것이죠. 그 결과 답변 품질 하락 없이 대답 속도를 기존 40초에서 15초대로, 무려 60% 단축하는 쾌거를 이루었습니다.
-->

# 🧠 6. 핵심 기술 3: 투트랙(Two-Track) LLM 전략

매핑 실패 시 방어를 포기하지 않고 최상위 두뇌 모델 가동.

- **기본 트랙**: 규정이 명확한 물품 ➡️ 빠른 텍스트 생성용 `GPT-5-mini` 모델 처리.
- **Fallback 트랙**: "보조배터리", "칼" 등 DB에 100% 매치되지 않으나 항공 보안 지식으로 유추가 필요한 일상 물품 처리.
- 🌟 **GPT-5.2 개입**: Fallback 트랙은 최고의 지능을 가진 플래그십 모델(`GPT-5.2`)로 우회 연결. DB를 간접 참고하며 글로벌 IATA 표준에 맞춰 기내/위탁 기준을 구체적으로 대리지시.

<!--
[대본]
세 번째 엣지는 '투트랙 폴백(Fallback) 구조'입니다. DB에 전혀 매핑되지 않는 모호한 물건들, 예를 들어 '칼'이나 '드라이기'를 묻는다면 챗봇이 무너져야 할까요? 아닙니다. 이 경우 저희 시스템은 가장 무겁고 똑똑한 플래그십 엔진인 GPT-5.2로 라인을 우회시킵니다. 세계 항공 표준 지식을 베이스로, 기내와 위탁의 세밀한 기준을 논리적으로 추론하여 유저를 끝까지 책임지고 안내합니다.

# 🛡️ 7. 사용자 경험(UX) 극대화: 엣지 케이스 완벽 방어

실제 팀 단위 QA를 거쳐 발견한 **4대 논리 허점(Bug) 100% 타파**.

1. **핀포인트 질의**: "나 고추장 가능?" ➡️ "어디 가시는 중인지 **출발/도착 국가**를 알려주세요."
2. **환각(Hallucination) 원천 차단**: 한국어로 말했다고 한국 출발로 상상 기입 방지(`null` 강제).
3. **실시간 환승(Dynamic Override)**: 대화 도중 "나 미국 말고 아까 그 중국으로 바꿀게" ➡️ 기존 슬롯을 무시하고 즉각 **노선 덮어쓰기 완료**. 편의성 고도화.
4. **미지원 노선 경고벽**: 지원하지 않는 '일본' 노선 질문 시, 즉각 위험 안내 최상단 출력.

<!--
[대본]
